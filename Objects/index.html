PART 1 — Beginner (JS Objects Basics)

Understand what objects are and how to use them.

🎯 Goal:

Understand what objects are and how to use them.

Topics Covered

  • What is an Object
  • Object literal syntax
  • Accessing properties (. vs [])
  • Add / update / delete properties
  • Nested objects
  • Looping through objects
  • Object methods
  • this keyword (basic)

Learning Example

const user = {
  name: "Sahadat",
  age: 25,
  greet() {
    return "Hello " + this.name;
  }
};

Key Concepts

  • Objects store related data
  • Functions inside objects are called methods
  • this refers to the current object

Practice Ideas

Create a car object

Define properties like make, model, year, and methods like start(), drive()

Create a student object with method getResult()

Include properties for name, grades, and a method to calculate results

Loop through an object and print key-value pairs

Use for...in loop to iterate through object properties

🟡 PART 2 — Intermediate (OOP Foundations)

Understand how JavaScript models objects and reuse behavior.

🎯 Goal:

Understand how JavaScript models objects and reuse behavior.

Topics Covered

  • Constructor functions
  • new keyword
  • Prototypes
  • __proto__
  • Object.create
  • Class syntax (class, constructor)
  • Inheritance (extends, super)

Learning Example

class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
  greet() {
    return `Hi, I'm ${this.name}`;
  }
}

Concepts

  • JS uses prototype-based inheritance
  • Classes are syntax sugar over prototypes

Practice

Create Animal → Dog → Cat hierarchy

Implement inheritance chain with base Animal class and derived Dog and Cat classes

Build a BankAccount class

Create a class with methods for deposit, withdraw, and getBalance

Override methods

Practice method overriding in inherited classes

🟠 PART 3 — Advanced (Real OOP + Internals)

Understand how JS actually works behind the scenes.

🎯 Goal:

Understand how JS actually works behind the scenes.

Topics Covered

  • Prototype chain deep dive
  • hasOwnProperty
  • instanceof
  • Object.getPrototypeOf
  • Encapsulation (private fields #)
  • Getters & setters
  • Composition vs Inheritance
  • Mixins

Learning Example

class Bank {
  #balance = 0;
  deposit(amount) {
    this.#balance += amount;
  }
  get balance() {
    return this.#balance;
  }
}

Concepts

  • Data hiding with private fields
  • Prefer composition over inheritance when possible

Practice

Create a logger mixin

Implement a mixin that adds logging capabilities to any class

Build a plugin system

Design a system that allows plugins to extend functionality

Write your own inheritance using prototypes

Manually implement inheritance without using class syntax

Practice

Implement Observer for notifications

Create a notification system using Observer pattern

Create plugin-based architecture

Design a system that supports plugins and extensions

Quick Reference

Object Literal Syntax

Class Syntax

Inheritance

  }
